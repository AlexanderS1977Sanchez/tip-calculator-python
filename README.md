# Tip Calculator (Python CLI)

A simple but well-structured **Tip Calculator** built with Python.  
This project takes a bill amount, a tip percentage, and optionally splits the total among multiple people.

It was designed as a beginner-friendly project with:
- Clear input validation
- Modular code (separate files for logic, input, display)
- Unit tests

---

## Features

✅ Enter bill amount (example: 25.50)  
✅ Enter tip percentage (example: 10, 12, 15)  
✅ Option to split the bill  
✅ Displays:
- Tip amount
- Total amount
- Amount per person (if split)

✅ Input validation:
- Bill must be > 0
- Tip must be >= 0
- People must be >= 1

✅ Unit tests included (using Python built-in `unittest`)
